INSTALL
=======
You need to install Xulrunner:

- Download:
ftp://ftp.mozilla.org/pub/mozilla.org/xulrunner/releases/1.8.0.1/linux-i686/en-US/xulrunner-1.8.0.1.en-US.linux-i686.tar.gz

- Install instruction (in French):
http://xulfr.org/wiki/XulRunner/Installation

Running
=======

xulrunner application.ini

To launch the application with a different locale:
xulrunner -UILocale fr-FR application.ini
